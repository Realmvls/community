# community
flask_project
由于教程版本问题以及个人水平问题该community暂停，过段时间重新编写
具体实现功能：
    用户管理，包括管理登录，会话，用户角色，权限以及用户头像。
    数据库管理，包括迁移处理。
    Web 表单支持，包括对各个字段的验证。
    分页处理。
    全文搜索。
    用户邮件提醒。
    HTML 模板。
    支持多国语言。
    缓存以及其它性能优化技术。
    开发以及生产服务器的调试技巧。
    在生产服务器上安装。
教程博客地址：http://www.pythondoc.com/flask-mega-tutorial/helloworld.html
